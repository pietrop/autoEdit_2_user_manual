# Install

To get all the features fulling working you need to have **Adobe Premiere version `12.1`** or greater.

{% hint style="info" %}
There are two ways to install the autoEdit2 panel. 

1. is through the [Adobe's Exchange marketplace](adobe-exchange.md). 
2. Install as third party extension, 
   1. [Using a third party package manager - Anastasiy](using-a-third-party-package-manager-anastasiy.md)
   2. [Via Terminal](via-terminal.md)
{% endhint %}
